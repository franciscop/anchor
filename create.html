# Create a dataset To create a dataset please follow these steps: - Create a new [Blank Google Sheet](https://docs.google.com/spreadsheets/u/0/) - Add two or three columns like this (the *extra* column is optional): | **word** | **meaning** | **extra** | |------------|---------------|-------------| | hola | hello | | | adiós | bye | | The first word will be the one shown by default and the second one the one displayed below (and the one to guess). The third one is the pronuntiation or an extra note you wanna write. - **Important**: publish it. File > Publish to the web... > **Publish** - Copy the url and paste it in the "Sheet url"
